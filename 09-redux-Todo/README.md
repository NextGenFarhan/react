# Redux Toolkit Todo App

A simple **Todo application** built with **React** and **Redux Toolkit**.  
You can **add**, **edit**, and **delete** todos with a clean UI styled using **Tailwind CSS**.

---

##  Features

-  Add new todos  
-  Edit existing todos  
-  Delete todos  
-  Global state management using Redux Toolkit  
-  Styled with Tailwind CSS  

---

## Tech Stack
- **React**
- **Redux Toolkit**
- **React Redux**
- **Tailwind CSS**
- **JavaScript (ES6+)**

---

## What is Redux Toolkit?
**Redux Toolkit (RTK)** is the **official, recommended way** to write Redux logic.  
It simplifies Redux by reducing boilerplate code and handling common tasks automatically.

---

## Applications of Redux Toolkit

Redux Toolkit is used when:
- Managing **global state** in large React apps
- Sharing data between multiple components
- Handling **complex state logic**
- Building apps like:
  - Todo apps
  - E-commerce carts
  - Authentication systems
  - Dashboards
  - Admin panels

---

## Why Use Redux Toolkit?

- Less boilerplate code  
- Faster development  
- Built-in best practices  
- Comes with useful tools:
  - `createSlice`
  - `configureStore`

---

## Key Benefits
- No need to write action types manually
- No switch-case reducers
- Immutable updates made easy
- Better performance and debugging
