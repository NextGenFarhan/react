# iTask Todo App
A simple React Todo application with add, edit, delete, and toggle functionality. Todos are also saved in local storage so that they persist between page reloads.

---

## Features
- Add new todos
- Edit existing todos
- Delete todos
- Mark todos as completed
- Todos are persisted in **localStorage**
- Simple and responsive UI

---

## Technologies Used
- **React** (Functional Components & Hooks)
- **Context API** for state management
- **Tailwind CSS** for styling
- **Local Storage** for data persistence
